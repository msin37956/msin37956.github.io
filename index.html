내용

프로젝트 개요

스마트시티 통합플랫폼 개발 및 TTA 스마트시티 통합플랫폼 인증 획득

작업기간

2021.5 ~ 2021.9

작업내용

  • UI/UX개발(화면스토리보드 일부 참여. 시설물관제, 이벤트 관제 화면 위주)
  • 화면 퍼블리싱

화면

스마트시티 통합플랫폼 로그인

시설물 모니터링. 등록된 시설물(cctv 등)의 목록과 지도상에서 시설물의 위치를 확인 가능

실시간 CCTV 영상 조회 팝업. 시설물 모니터링에서 시설물 마커를 선택하면
해당 CCTV의 영상 확인 및 CCTV의 PTZ 제어가 가능

이벤트 모니터링. 발생한 이벤트의 목록과 지도상에서 이벤트 발생의 위치를 확인 가능

이벤트 상세. 이벤트 모니터링에서 특정이벤트를 선택하여 개별 이벤트 관제가 가능하며,
이벤트 관제시 사건 발생지점에서 가장 가까운 시설물(cctv)5개의 목록과 영상화면을 자동으로 표출

데이터 조회. 119긴급출동 이벤트 발생 이력 조회 가능

상세정보 팝업. 데이터 조회 페이지에서 화면에서 모든 데이터를 표출할 수 없는 경우
테이블의 상세정보 버튼을 눌러 상세정보 팝업으로 데이터 전문 확인 가능

내부iot 가전기반 스마트홈 실증형 기술 개발 사업 과제
(과제명:스마트시티 연계 취약계층을 위한 커뮤니티케어 기반 라이프케어 서비스 개발 및 실증)

내용

프로젝트 개요

세대 내 iot 가전을 통해 수집된 데이터를 통한 생활패턴 분석과 실증 세대의 상태를 모니터링하고 응급상황 발생시 알람을 발생시키는 모니터링 시스템

작업기간

2020 ~ 2021

작업내용

  • 화면설계(웹/모바일웹)
  • UI/UX개발
  • 화면 퍼블리싱

화면

스마트홈 통합모니터링시스템 로그인

대시보드. 왼쪽은 실시간 정보를 표시하며(실증세대의 상태, 오늘 누적이벤트, 실시간 이벤트), 오른쪽은

대시보드-응급알람팝업. 응급알람이 발생시 화면 정중앙에 팝업을 뛰움

개별모니터링. 원하는 세대를 선택해 해당 세대의 실시간 상태를 확인 가능

단지별 에너지 모니터링. 단지별 이번달/지난달 에너지 사용량을 확인 가능

데이터 조회. 이벤트 발생, iot 가전을 통해 수집된 데이터, 장비오류 발생에 대한 데이터 조회 가능

단지별 월별 에너지 통계. 단지별 월별 에너지 통계 확인 가능

스마트홈 통합모니터링시스템 모바일웹. 대시보드(왼쪽), 대시보드-이벤트 리스트 펼침(오른쪽)

모니터링. 그룹보기(왼쪽)-단지별(그룹별) 모니터링, 상태모아보기(오른쪽)-상태별(응급/위험/주의/통신오류) 모니터링

개별 세대 모니터링. 상세상태(왼쪽), 그래프(중간), 세대알림이력(오른쪽)

기타 페이지. 이벤트조회(왼쪽), 실증세대 설치현황(중간), 내정보(오른쪽)

납품온압보정기 모니터링 시스템

내용

프로젝트 개요

대용량 가스를 사용하는 장소(빌딩, 학교, 식당, 공장 등)에 설치된 온압보정기를 통해 가스 사용량의 자동검침 및 데이터 전송을 통해 검침 데이터의 모니터링 및 검색, 리포팅 서비스를 제공

작업기간

2020

작업내용

  • UI/UX개발(대시보드 화면설계, 기타화면은 스토리보드 작업자 별도)
  • 화면 퍼블리싱

화면

미터링크(온압보정기 모니터링 시스템) 로그인

대시보드. 왼쪽의 사이드메뉴의 트리뷰에서 업체를 선택해 업체에 등록된 장비의 상태와 가스사용량을 확인 가능

장비상세정보 팝업. 개별 장비 박스를 선택하면 상세정보 팝업에서 장비의 상세정보를 확인 가능

그룹조회. 왼쪽 사이드메뉴에서 조건을 선택에 해당하는 장비 검색 가능

데이터 조회. 등록 장비별 데이터 조회 화면

미터링크 모바일 웹앱. 모바일웹앱은 웹사이트 버전을 축소시켜 모니터링에 특화
